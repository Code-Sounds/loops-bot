# Loops Bot

Bot para o discord, com finalidade de tocar músicas em um canal de voz em loop, 24 horas por dia.

## Comandos

- **[!loop]** - Inicia o loop de músicas no novo servidor.
- **[!stop]** - Parar o bot no servidor.
